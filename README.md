# Book Recommendation System

Hybrid Recommendation System combining Content-based Filtering (KNN) and Collaborative Filtering (SVD) for predicting and recommending books to users.

## Project Structure
- `data/`: Contains the goodbooks-10k extended dataset.
- `notebooks/`: For Exploratory Data Analysis (EDA) and experimental code.
- `src/`: Core Python modules.
- `main.py`: Entry point for demo.
